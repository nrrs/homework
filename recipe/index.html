Description

A BLT (Bacon, Lettuce, and Tomato) is a type of bacon sandwich. Wiki


Ingredients

  • Bacon
  • Lettuce
  • Tomato
  • Bread

Directions

  1. Place bread on plate.
  2. Add Bacon.
  3. Add Lettuce.
  4. Add Tomato.
  5. Optional: Add basil, add cheese, and grill.
